HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element is not visible to the client or user
  • The body element uses h1, h2, h3... h6, to identify specific headers in the body or on the webpage
  • HTML acts as the metaphorical frame or foundation of the code
  • It's best to save and test code as often as possible to make sure there are no errors

CSS

  • CSS acts as the metaphorical physical style elements of the code when viewed in a web browser such as google
  • When you open a live webpage in google you can use the inspect element tool to inspect the different elements of the html and css code
  • Can help you understand the viewport; what the website will look like on different devieces and aspect ratios
  • Can help to dictate what certain text will do when webpage is shrunken and expanded, shows how to wrap text, what the website should do for certain triggers
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When you list dimensions its going to apply to the X and Y axes respectively
  • You can use hex codes or just use namming semantics to get a desired or specific colour
  • Remember to link all your style sheets with your html in the header but at the very bottom of all the meta
  • if you specify your classes and your ids, you can manipulate them all at the same time in your style css

Git

  • Git is the online cloud based repository that you push and pull your code from
  • You can work on code independent from others when you use a branch so that you dont caue conflict with the main branch
  • you can send stored code to and inspector before pulling it down into the main branch
  • It is important to pull all changes in first before doing any work so that you can stay up to date and code won't conflict
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
